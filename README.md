# Beta-Rec | Discord Bot
Beta-Rec is a Discord bot designed for the Roblox game Boxing Beta. It helps communities manage and track fighters' boxing careers by recording professional fight records similar to BoxRec.

## Purpose 
Beta-Rec is built for Roblox Boxing Beta leagues, organizations that want a professional way to keep track of fighters and present their records with clean.
Whether you're running a small league or a large championship league, Beta-Rec simplifies record keeping and gives your fighters a polished, professional profile.
